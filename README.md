# Calendar-Extension


## COMANDOS GITS: faça toda vez que realizar alguma alteração
- Criar branch: `git checkout -b NOME_BRANCH`
- Atualizar Branch: `git pull origin main`
- Preparando a branch para commit (Pre commit): `git add .`
- Commitando: `git commit -m "TEXTO PARA COMMIT"`
- Publicando branch:`git push -u`
