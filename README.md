# 📅 Calendar Alert App

Aplicação composta por:
- API REST (Node.js)
- Frontend Web (Angular)
- Extensão de Navegador (Chrome/Edge)

## 🚀 Objetivo
Gerenciar atividades e enviar notificações quando estiverem próximas.

## 🧱 Estrutura
- backend: API Node.js
- frontend: Aplicação Angular
- extension: Extensão do navegador
- database: Scripts SQL Server
- docs: Documentação
